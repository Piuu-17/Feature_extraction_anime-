 Anime Feature Extraction & Analysis Project

Project Description

This project focuses on feature extraction and basic data analysis using Python and Pandas on an anime dataset. The notebook extracts meaningful information such as the number of episodes and total runtime from raw text data in the `Title` column, creates new structured features, and performs simple exploratory analysis to identify top-rated and longest-running anime.

The project demonstrates:

* Data cleaning and preprocessing
* Feature extraction from text columns
* Working with datasets using Pandas
* Basic exploratory data analysis (EDA)
* Creating new columns from unstructured text
* Filtering and analyzing data

This is a beginner-friendly data analysis project suitable for learning:

* Python
* Pandas
* Data preprocessing
* Exploratory Data Analysis (EDA)
* String manipulation



 Technologies Used

* Python
* NumPy
* Pandas
* Jupyter Notebook



Dataset

The dataset used in this project is:

* `anime.csv`

The dataset contains anime-related information such as:

* Anime title
* Episode count
* Ratings/Scores
* Runtime details






The project can:

* Extract episode counts from anime titles
* Generate cleaned datasets
* Display top-rated anime
* Find longest-running anime series
